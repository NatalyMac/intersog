Клиент занимается организацией конференций и хочет сделать сеть для общения клиентов

Пользователи:
-админы - могут все
-компании/спонсоры - могут делать спонсорство для конференций (загружают данные о себе, лого), могут добавлять себе представителей из числа докладчиков, тогда в профиле докладчика отображается лого при просмотре
-слушатели/докладчики - могут регистрироваться и общаться между собой (портрет, данные о себе)

Разделы:
-личный профиль
 -настройки
 -почта (получателю отправляется дублирующее мыло, что "вам написали")

-текущие конференции
 -список компаний-спонсоров
 -регистрация для слушателей и докладчиков
 -список докладчиков
 -после окончания конференции, слушатели могут посмотреть список профилей других слушателей и написать им

-архив с конференциями (в архив уходит спустя 3 дня после окончания)

-новостная лента
